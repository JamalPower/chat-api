AI Chat API
Overview

Generate text responses using multiple AI providers. Supports Gemini, Groq, OpenRouter, and Mistral with automatic fallback to available models if primary fails.

Endpoint
POST
/api/models/reply
Request Parameters
Parameter Type Required Description
provider string ✓ AI provider: "gemini", "groq", "openrouter", "mistral"
text string ✓ Your prompt/message to send to the AI
model string ✗ Specific model ID (uses provider default if omitted)
Request Example
fetch('/api/models/reply', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ provider: 'gemini', model: 'gemini-1.5-flash', text: 'Explain quantum computing' }) }) .then(res => res.json()) .then(data => console.log(data.response))
Response Example
{ "response": "Quantum computing is a revolutionary...", "provider": "gemini", "model": "models/gemini-1.5-flash", "modelName": "Gemini 1.5 Flash", "fallbackUsed": false, "attempts": [] }
Available Models by Provider
Provider Model ID Description
gemini models/gemini-1.5-flash Fast, versatile multi-turn chat model
gemini models/gemini-1.5-pro High-capacity, complex reasoning
groq llama3-8b-8192 Fast Llama 3 8B on LPU
groq mixtral-8x7b-32768 High-performance mixture of experts
mistral open-mistral-7b General purpose Mistral model
openrouter meta-llama/llama-3-8b-instruct:free Free Llama 3 instruct model
💡 Tip: If your selected model/provider fails, the API automatically tries fallback models from the priority queue until one succeeds.
Game Search API
Overview

Search for games, get autocomplete suggestions, and fetch detailed game information including ratings, platforms, developers, and play statistics.

Search Games
GET
/api/scrap/games/list
Parameter Type Description
query string Game title to search for (URL encoded)
type string Response type ('JSON' or 'HTML')
page number Page number for pagination (default 1)
fetch('/api/scrap/games/list?query=Elden Ring') .then(res => res.json()) .then(data => { console.log(data.data); // Array of games // [{ name, img, year, type, url }, ...] })
Autocomplete Suggestions
GET
/api/scrap/games/autocomplete
Parameter Type Description
query string Partial game name (min 2 chars)
fetch('/api/scrap/games/autocomplete?query=El') .then(res => res.json()) .then(data => console.log(data.html)) // Returns HTML with autocomplete items
Game Details
GET
/api/scrap/games/details
Parameter Type Description
url string Game URL from search results (URL encoded)
fetch('/api/scrap/games/details?url=/games/elden-ring') .then(res => res.json()) .then(data => { const game = data.data; // { title, coverImage, rating, developers, // publishers, platforms, genres, playTimes, ... } })
📍 Source: Data is scraped from backloggd.com, a gaming tracking platform with ratings and statistics.
Web Scraper API
Overview

Fetch HTML source code from any website. Useful for web scraping, data extraction, and inspecting page structure with proper User-Agent spoofing to bypass some restrictions.

Endpoint
GET
/api/scrap/fetch
Request Parameters
Parameter Type Required Description
url string ✓ Full URL to fetch (must include https://)
Request Example
fetch('/api/scrap/fetch?url=https://example.com') .then(res => res.text()) .then(html => { console.log(html); // Complete HTML source // Parse with DOMParser or use with libraries })
Response

Returns the complete HTML source code as plain text. Parse with DOMParser or extract data using libraries like Cheerio (Node.js) or BeautifulSoup (Python).

⚠️ Note: May fail on sites with strong anti-bot protections (Cloudflare, etc.). Respects robots.txt and standard web scraping etiquette.
Integration Tips & Examples
Python Example
import requests # Chat API response = requests.post( 'https://chat-api-azure.vercel.app/api/models/reply', json={ 'provider': 'gemini', 'text': 'Hello, how are you?', 'model': 'gemini-1.5-flash' } ) print(response.json()['response']) # Game Search games = requests.get( 'https://chat-api-azure.vercel.app/api/scrap/games/list', params={'query': 'Elden Ring'} ).json()['data']
Node.js Example
const BASE_URL = 'https://chat-api-azure.vercel.app'; // Chat API const chatResponse = await fetch(`${BASE_URL}/api/models/reply`, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ provider: 'groq', text: 'Write a hello world in JS', model: 'llama3-8b-8192' }) }); const data = await chatResponse.json(); console.log(data.response); // Game Details const gameDetail = await fetch( `${BASE_URL}/api/scrap/games/details?url=/games/elden-ring` ).then(r => r.json());
Error Handling
// Check for errors in response fetch('/api/models/reply', { method: 'POST', body: JSON.stringify({ provider, text, model }) }) .then(res => { if (!res.ok) throw new Error(`HTTP ${res.status}`); return res.json(); }) .then(data => { if (data.error) { console.error('API Error:', data.error); // Check data.attempts for fallback info if (data.attempts) { data.attempts.forEach(a => console.log(`Failed: ${a.provider} - ${a.error}`) ); } } else { console.log(data.response); } }) .catch(err => console.error('Network error:', err));
